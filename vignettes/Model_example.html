Example of modelling the EQ-5D score and visual analogue scale

Adrian Barnett

2020-12-22

This vignette shows how to use the lmvas function to model EQ-5D quality of life data.

Example data

We use an example data set from a survey of people with and without lower limb osteoarthritis in Spain [1]. We load the eq5d.plus library which includes a random sample of 1,000 participants from the Spanish osteoarthritis study in a data set called arthritis.

library(eq5d.plus)
summary(arthritis)
#>      sex                age               EQ5D_VAS           EQ5D

The data set has just four variables with the two EQ-5D measures, sex and age.

We visually examine the effect of age using box-plots.

# libraries needed for plotting and data management:
library(ggplot2)
library(tidyr)
library(stringr)
suppressPackageStartupMessages(library(dplyr))
# switch to long format:
long = gather(arthritis, `EQ5D_VAS`, `EQ5D`, key='type', value='eq') %>%
  mutate(type = ifelse(str_detect(type, 'VAS'), "VAS", "Score"))
# plot:
boxplot = ggplot(data=long, aes(x=age, y=eq))+
  geom_boxplot(fill='darkseagreen2')+
  facet_wrap(~type)+
  xlab('Age group')+
  ylab('EQ-5D')+
  theme_bw()
boxplot

The plot shows that older age groups have a generally lower quality of life for both the score and VAS. The VAS has generally lower levels than the score.

Linear model

Below we run a linear model. The model is run in a Bayesian paradigm using the INLA library.

model = lmvas(vas_name='EQ5D_VAS', eq_name='EQ5D', independent_vars = c('age','sex'), data=arthritis)
summary(model)

The model shows the clear decrease in quality of life with older age. The oldest age group of 85+ has a mean quality of life that is 0.216 lower than the youngest age group of 50 to 54 years (the reference group for age). The 95% credible interval for this difference is from -0.269 to -0.163.

There is also a lower mean for the VAS compared with the score of 0.153 (95% credible interval: -0.164 to -0.141).

The EQ-5D score and VAS are positively correlated with an estimated correlation of 0.557 (“rho” estimate in model hyperparameters).
